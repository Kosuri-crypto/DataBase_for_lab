# Вступление
Интерфейс подготовлен для лабораторной работы по курсу *"базы данных"*.
***
Программа создана при помощи ***Unity*** с использованием ***SQLite***.
Сама база данных создана при помощи СУБД ***DB Browser for SQLite***.
